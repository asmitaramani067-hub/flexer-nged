# Garet Font Files

This directory should contain the Garet font files:

- `garet-book.woff2` (Weight 400)
- `garet-heavy.woff2` (Weight 800)

## How to Add Fonts

1. Obtain the Garet font files (`.woff2` format recommended for best performance)
2. Replace the placeholder files in this directory with the actual font files
3. Ensure the filenames match exactly:
   - `garet-book.woff2`
   - `garet-heavy.woff2`

## Font Licensing

Make sure you have the proper license to use the Garet font in your project.

## Fallback

If the Garet fonts are not available, the application will fall back to system sans-serif fonts. The Dosis font (from Google Fonts) will still load for body text.
